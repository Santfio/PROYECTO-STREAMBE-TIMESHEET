# Timesheet App

Bienvenido/a al Timesheet App, una aplicación web diseñada para registrar y organizar tus horas de trabajo de forma rápida, clara y accesible.  
La idea principal de este proyecto es ofrecer una herramienta sencilla para que cualquier persona o equipo pueda llevar un control básico de tiempo sin depender de hojas de cálculo ni configuraciones complicadas.

Timesheet App te permite iniciar sesión con una cuenta de demostración, cargar tus horas trabajadas indicando la fecha, el proyecto y una breve descripción, y consultar o eliminar los registros cuando sea necesario.  
Todo se guarda temporalmente en el servidor, pensado para un entorno de prueba o demostración, sin necesidad de crear cuentas reales.

Podés probar la aplicación usando la siguiente cuenta de acceso:  
**Email:** demo@example.com  
**Contraseña:** password  
Solo necesitás hacer clic en “Autocompletar demo” o escribir los datos manualmente para empezar a explorar el sistema.

Una vez dentro, el funcionamiento es muy simple: iniciás sesión, completás los campos de fecha, proyecto y cantidad de horas, y luego podés revisar o eliminar tus registros en cualquier momento.  
La interfaz está pensada para ser intuitiva, minimalista y usable tanto en computadora como en dispositivos móviles.

El proyecto fue desarrollado con **React** en el frontend y **Node.js + Express** en el backend, y se encuentra desplegado en línea utilizando **Vercel** y **Render**, lo que permite acceder desde cualquier lugar sin necesidad de instalación local.  
El objetivo principal fue practicar la integración entre frontend y backend, mejorar el manejo de datos con API REST y lograr un diseño funcional y fácil de mantener.

Timesheet App es un proyecto personal creado con fines educativos, como parte del proceso de aprendizaje y consolidación de habilidades en desarrollo web.  
Está disponible públicamente para quienes quieran probarlo, revisarlo o tomarlo como referencia.
